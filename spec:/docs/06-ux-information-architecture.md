## Roadmap workspace layout

### Left rail
- roadmap selector
- saved views
- filters
- theme / team grouping toggles

### Main canvas
- timeline view
- zoom controls
- date range controls
- swimlane grouping
- drag-and-drop roadmap bars and milestones

### Right panel
- selected initiative / roadmap item detail
- AI actions
- comments / history
- linked Jira / Confluence artifacts

## Design principles
- table + timeline duality
- detail without modal overload
- always-visible filters
- fast edit from side panel
- consistent color system
- executive mode vs editor mode
